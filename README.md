### Commit Rule

- 문제 풀이 성공시
    - Solved : 풀이사이트 문제명
    - **Ex)** Solved : Programmers 명예의 전당
- 문제 풀이 실패시
    - Failed : 풀이사이트 문제명
    - **Ex)** Failed : Programmers 명예의 전당

못 풀면 풀이한 곳까지만 올리시면 됩니다~

### File name Rule

- 풀이사이트_문제명_이름
- **Ex)** Programmers_명예의 전당_영원
